## 📝 使用建议

### 配色方案切换
推荐使用命令行快速切换：
```bash
./switch-theme.sh blue     # 蓝色科技风
./switch-theme.sh orange   # 橙红渐变风  
./switch-theme.sh green    # 绿色清新风
./switch-theme.sh purple   # 恢复紫色（默认）
```

或者打开 `theme-preview.html` 在浏览器中预览所有配色方案。

- **主应用**: http://localhost:8080
